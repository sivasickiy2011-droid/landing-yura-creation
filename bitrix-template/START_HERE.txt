═══════════════════════════════════════════════════════════════════
  БЫСТРАЯ УСТАНОВКА (ДЛЯ ОПЫТНЫХ)
═══════════════════════════════════════════════════════════════════

  1. Скопировать папку в /local/templates/bitrix24_partner/
  2. Активировать шаблон в настройках сайта
  3. Создать 4 инфоблока (capabilities, pricing, reviews, news)
  4. Обновить ID инфоблоков в index.php
  5. Создать веб-форму заявок
  6. Настроить верхнее и нижнее меню
  7. Загрузить изображения в /images/
  8. Очистить кеш

  ⏱️ Время: ~15 минут

═══════════════════════════════════════════════════════════════════
  НУЖНА ПОМОЩЬ?
═══════════════════════════════════════════════════════════════════

  1. Проверьте права на файлы (755 для папок, 644 для файлов)
  2. Очистите кеш Битрикс (Настройки → Производительность)
  3. Проверьте логи (/bitrix/modules/main/classes/general/event_log.php)
  4. Откройте консоль браузера (F12) для поиска JS-ошибок

  Если не помогло:
    → Партнерская поддержка Битрикс24
